HTML5Apps Platform

Fully featured web apps platform, that can bring Floflis as a graphical app to any OS.

It is like PortableApps Platform. It can run Floflis Central and any other Electron app, but using the same runtime: so apps which had thousand megabytes will have just a few kilobytes.


Yet in development. See it on GitHub or take a glance at its in-development site.
